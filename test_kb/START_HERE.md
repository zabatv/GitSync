# База знаний экосистемы

Добро пожаловать! Это единая цифровая память нашей экосистемы.

## Структура

- **00_core** — миссия, философия, принципы
- **01_products** — 4 продукта: IThub, Фора, Перемена, Инкубатор  
- **02_entities** — справочники и реестры
- **03_shared_services** — общие сервисы
- **04_ai_system** — AI-агенты и промпты
- **05_templates** — шаблоны документов
- **06_archive** — архив
- **07_sandbox** — песочница для черновиков

## Как работает

1. Все изменения в Obsidian автоматически синхронизируются
2. AI может читать документы через RAG
3. Структура единая для всех продуктов
