# GitSync - Автоматическая синхронизация

## Установка (для пользователей)

### Шаг 1: Установите Git
Скачайте и установите: https://git-scm.com/download/win

При установке выберите:
- "Git from the command line and also from 3rd-party software"
- "Use the default options"

### Шаг 2: Настройте папку

1. Создайте папку для проекта (например `C:\Projects\MyProject`)
2. Скопируйте в неё файл `start.bat` из этого архива
3. Откройте папку в проводнике
4. Правой кнопкой → "Open Git Bash here"
5. Выполните:
```bash
git clone ВАШ_РЕПОЗИТОРИЙ .
```

Пример: `git clone https://github.com/ваша_компания/проект.git`

### Шаг 3: Запустите

Дважды кликните по `start.bat`

**Готово!** Синхронизация работает автоматически каждые 5 минут.

---

## Как это работает

- `start.bat` автоматически делает `git pull` каждые 5 минут
- Если вы внесли изменения — сначала сделайте `git push` вручную
- Скрипт работает в фоне даже после закрытия окна

## Требования

- Windows 7 или выше
- Git (см. шаг 1)
- Интернет-соединение
